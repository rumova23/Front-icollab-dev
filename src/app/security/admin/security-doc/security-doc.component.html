Tablas

Así seávisualizaran la tabla

<app-mat-table 
	[data]="tableDataDemo"
	(clickSee)="tableRowSee($event)"
	(clickEdit)="tableRowEdit($event)"
	(clickDelete)="tableRowDelete($event)"
></app-mat-table>

--Propiedades obligatorias--

[data]="tableDataDemo"

Esta tabla recibe una lista de objetos simples, a través de la propiedad [data] (no acepta objetos con propiedades anidadas).
Al alterar la lista de objetos en el componente “padre”, la tabla detecta el cambio y se actualiza automáticamente.
Si no se especifican las demas listas seran generadas automaticamente.

--Eventos--

Esta tabla emite tres eventos diferentes:

(clickSee)="tableRowSee($event)": Para ver un registro.

(clickEdit)="tableRowEdit($event)": Para editar un registro.

(clickDelete)="tableRowDelete($event)": Para eliminar un registro.

--Propiedades opcionales--

[columnsLabels]="tablaColumnsLabels"

Con “[columnsLabels]” se podrá remplazar el texto en el “header” de cada columna. Este es una lista de objetos donde se especifica el “key” y el nuevo “label” , el “key” tiene que coincidir con en nombre de la propiedad en el listado de datos principal “[data]”, tambien indica que columnas seran craedas.

[columnsDisplay]="tableColumnsDisplay"

Es una lista de "string" donde se especifican las columnas que seran visibles en la tabla, estas tienen que ser las mismas que el nombre de la propiedad de la lista de datos principal.

Existen 3 columnas que genera el componente de forma automatica (estas columnas seran agregadas si se especifican en la lista "[columnsDisplay]="tableColumnsDisplay""):

“sys_see”, genera la columna con el botón para ver los detalles del registro.

“sys_edit”, genera la columna con el botón para editar el registro.

“sys_delete”, genera la columna con el botón para eliminar el registro.

[row_x_page]="tableRow_x_page"

Es un listado de números para poder seleccionar cuantos registros se mostrarán por página de la tabla

<app-mat-table 
	[data]="tableData"
	[columnsLabels]="tablaColumnsLabels"
	[columnsDisplay]="tableColumnsDisplay"
	[row_x_page]="tableRow_x_page"
	(clickSee)="tableRowSee($event)"
	(clickEdit)="tableRowEdit($event)"
	(clickDelete)="tableRowDelete($event)"
></app-mat-table>
