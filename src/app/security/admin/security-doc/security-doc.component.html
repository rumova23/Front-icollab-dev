Tablas

Así seávisualizaran la tabla

--Eventos--

Esta tabla emite tres eventos diferentes:

(clickSee)="tableRowSee($event)": Para ver un registro.

(clickEdit)="tableRowEdit($event)": Para editar un registro.

(clickDelete)="tableRowDelete($event)": Para eliminar un registro.

--Propiedades--

[data]="tableDataDemo"

Esta tabla recibe una lista de objetos simples, a través de la propiedad [data] (no acepta objetos con propiedades anidadas).
Al alterar la lista de objetos en el componente “padre”, la tabla detecta el cambio y se actualiza automáticamente.
Si no se especifican las demas listas seran generadas automaticamente.

[columnsLabels]="tablaColumnsLabels"

Con “[columnsLabels]” se podrá remplazar el texto en el “header” de cada columna. Este es una lista de objetos donde se especifica el “key” y el nuevo “label” , el “key” tiene que coincidir con en nombre de la propiedad en el listado de datos principal “[data]”.

[columnsDisplay]="tableColumnsDisplay"

Es una lista de "string" donde se especifican las columnas que seran visibles en la tabla, estas tienen que ser las mismas que el nombre de la propiedad de la lista de datos principal “[data]”.

[showPagination]="false"

Este atributo puede usarse si desea ocultar el paginador

selectionLabel="Visible"

Si se desea darle un titulo a la columna que contiene los checkbox se le puede paras el titulo mediante el atributo 'selectionLabel'

[selection]="selection"

Es una lista de "objetos identicosa [data]" que representan los elementos con el checkbox seleccionado.

{{codes.tableSelectionModel}}

esto retornara una lista con los elementos que se encuentran seleccionados.

Existen 4 columnas que genera el componente de forma automatica (estas columnas seran agregadas si se especifican en la lista "[columnsDisplay]="tableColumnsDisplay""):

“sys_checkbox”, genera la columna con un checkbox por fila.

“sys_see”, genera la columna con el botón para ver los detalles del registro.

“sys_edit”, genera la columna con el botón para editar el registro.

“sys_delete”, genera la columna con el botón para eliminar el registro.

[row_x_page]="tableRow_x_page"

Es un listado de números para poder seleccionar cuantos registros se mostrarán por página de la tabla

Botones

{{codes.btn}}

A los botones se les puede indicar su tipo "Type" mediante el atributo "myType", este atrubuto por defecto es de tipo "button", pero se puede especificar para que sea de tipo "submit".

Tiene su propio evento "click" que se llama mediante su propiedad "myClick" , esto con el fin de que si el boton se desactiba no se lanza el evento.

mediante la propiedad "disabled" se le puede indicar que desactive el boton.

si dentro del boton requieres un texto muy particular que no este dentro de los botones que se utilizan , se puede hacer uso del atributo "label" con el componente {{codes.btnBase}}
